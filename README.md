# Project-04-Nodejs-Express-App
# Author: Jordan Neslony

# Description:
This website is an app aims to provide users with questions on the most hit and recent games, whenever they have registered and logged into an account.

# Hosting Link: https://project-04-nodejs-express-app.onrender.com/

# Functionality: 
- This project features a nav-bar at the top of the screen that is navigatable
 via hyperlinks.
- This project features a dashboard that is navigatable by clicking buttons in the left column. Upon clicking a button it will display questions in the right panel.
- The main feature of this website lies in its frontend and backend communication. The site will require users to register and on their next visits log in to access the dashboard.
- The website has many different buttons that link or cause requests that link to many pages.
- Additionally this website is accesible for mobile devices and lower screen resoulution.

# List of Container and Presentational Components: 
1. Main Renderer
2. App Root
3. Home Screen
4. Login Screen
5. Registration Screen
6. Dashboard Screen
4. Dashboard List Item


# Technologies Used: HTML, CSS, SCSS, Vite + React, Babel, Render(for hosting the site), JavaScript, React Router DOM, MongoDB, NodeJS, Express, and Bootstrap.

# User Stories:
1. As a gamer, I want to know if there are any new video games in my favorite genre, so that I can experience playing them in the future.
2. As a game journalist, I want to know what video games people are interested in, so that I can provide accurate reviews on the top hit games.
3. As a game developer, I want to know what the hit genres are, so that I can make a game that appeals to my audience.
